# TripOS - Premium AI Travel SaaS

This project is a high-fidelity frontend implementation of TripOS, an AI-powered travel planning platform. It features a premium  design aesthetic with glassmorphism, smooth animations, and a dark futuristic theme.

## Tech Stack
- **Framework**: React + Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: CSS Transitions & Keyframes

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    ```bash
    npm run build
    ```

## Features
-   **Authentication**: Login page with immersive background.
-   **Trip Creation**: specific form for generating itineraries.
-   **Timeline**: Vertical itinerary visualization.
-   **Budget**: Interactive budget tracking with progress bars.
-   **Map**: Mock map interface with floating UI.
-   **Profile**: User stats and history.
-   **Settings**: Application preferences.

## Design Decisions
-   **Glassmorphism**: Heavy use of `backdrop-blur` and semi-transparent backgrounds to create depth.
-   **Gradients**: Custom `blue-cyan` gradients for primary actions and accents.
-   **Typography**: Clean sans-serif (Inter/SF Pro) for readability.
-   **Dark Mode**: Default and only mode for a premium feel.
